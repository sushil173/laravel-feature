## Getting Started

- Copy .env.example to .env
- Manage DB Settings, then run
```
php artisan migrate:refresh --seed
```
- [Create account on mailtrap](https://mailtrap.io/).
- Choose Laravel7+ Integrations and copy code to .env under mail settings.
- Ready to Rock. Then Run. 
```
php artisan serve
```
